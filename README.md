# Projetos 2 — Biogás (codigestão anaeróbia)

Projeto acadêmico **Projetos Industriais II (UFMS)**: produção de biogás a partir de **lodo de esgoto + vinhaca + resíduo orgânico**, com balanços de massa e energia, dimensionamento, otimização, análise econômica e **dashboard interativo** (`Dashboard_Biogas_Interativo.html`).

## Conteúdo principal

- Documentação em Markdown (`00_` … `12_`, entregas consolidadas)
- `Planilha_Biogas_Codigestao.xlsx` e scripts PowerShell `gerar_planilha_biogas.ps1` / `validar_planilha_biogas.ps1`
- PDFs de apoio (aulas e cronograma)
- Pasta `Penicilina_G_arquivado/`: tema anterior arquivado

## Como usar o dashboard

Abra `Dashboard_Biogas_Interativo.html` no navegador ou publique em servidor estático.
