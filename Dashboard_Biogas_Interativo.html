Simula uma planta de biogas por codigestao de lodo de ETE (Los Angeles + Imbirussu), vinhaca e residuo organico, com base de 500 m3/dia. Clique em ⚙ Premissas no topo para ajustar parametros.

1. Digestao Anaerobia

Decomposicao de materia organica na ausencia de O2 em 4 etapas: Hidrolise, Acidogenese, Acetogenese, Metanogenese. Produtos: biogas (CH4+CO2) e digestato.

2. Codigestao

Misturar substratos equilibra C/N. Lodo agregado (80%), Vinhaca (10%) e Residuo (10%) seguem o memorial da equipe para manter estabilidade biologica e coerencia com as correntes do PFD.

Diagrama Sankey — Fluxo de Massa

📚 Como ler este balanço

  • Conservação de massa: tudo que entra DEVE sair (regime permanente). Se o erro ≠ 0, há inconsistência.
  • SV → Biogás: Somente sólidos voláteis (SV) são convertidos. SF (sólidos fixos) saem intactos no digestato.
  • Scrubber: η=97% de remoção de CO₂; 3% de perda de CH₄ na água.
  • Flash: recupera 85% do CH₄ dissolvido (reciclo ao scrubber).
  • Stripper: regenera a água liberando CO₂ para atmosfera.

📚 Conceitos-chave deste balanço

PCI (Poder Calorífico Inferior): Energia liberada na combustão sem condensar H₂O. PCI_CH₄ = 35.8 MJ/Nm³.
BEM (Balanço Energia Mecânica): W = ρ·g·Q·H / η. Aplica-se a cada bomba para calcular potência de eixo.
Compressão Politrópica: W = P₁·V̇·(n/(n-1))·[(P₂/P₁)^((n-1)/n) - 1]. n=1.3 para biogás.
Perdas térmicas: Q_loss = U·A·ΔT. U≈1.5 W/m²K para concreto isolado. A = superfície do digestor.
Eficiência global: E_produto / E_química. Quanto do potencial químico do CH₄ se converte em biometano útil.
Referências: Perry’s 8ª ed.; Smith, Van Ness & Abbott Cap.2; Aula 16 Profa. Janaína (BEM).

Notas de Projeto

  • P-106 domina (>95% da potência) por conta do ΔP = 9 bar do scrubber HPWS.
  • P-104 opera em regime LAMINAR (Re<600) — fisicamente justificado: resíduo orgânico 15% ST, μ=80 cP. Bomba Moineau obrigatória.
  • P-108 tem Hgeo negativo (-8.24 m): escoamento por gravidade possível. Bomba instalada como reserva.
  • P-107: na prática, pressão da coluna (10 bar) pode dispensar bomba — válvula redutora suficiente.
  • Viscosidades revisadas conforme Metcalf & Eddy (2014) Tab.14-13 e pipeline v4 (μmix=5 mPa·s).
  • Vazao P-106/P-107: 81.63 m³/h = resultado direto do HPWS v4 (Qagua = 1959 m³/d).

🛑 Valvulas de Processo — Especificacao e Dimensionamento

Criterio: ISA S75.01 (Cv), API 6D (gaveta/esfera), ASME B16.34. Velocidade maxima: 3 m/s (liquido), 20 m/s (gas). dP projeto: 10-25% dP total da linha.

Cv = Q·√(SG/ΔP)  |  K = 890·d&sup4;/Cv²  |  Critério cavitacao: ΔP < FL²(P1-Pv)

Criterios de Selecao

  • Gaveta (XV-): ON/OFF, baixo dP, servico lodo/lama. API 6D.
  • Esfera (XV-): ON/OFF rapido, gas comprimido, vedacao total. API 6D.
  • Borboleta (BV-): Regulacao grosseira, grandes diametros, baixo custo.
  • Globo (FCV-): Controle fino de vazao, loop PID. ISA S75.01.
  • Retencao (CV-): Impedir refluxo, obrigatorio apos bombas. API 6D.
  • Seguranca (PSV-): Alivio de pressao, protecao de equipamentos. ASME VIII / API 520.
