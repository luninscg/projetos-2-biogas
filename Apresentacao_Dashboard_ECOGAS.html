1. O que é este Dashboard?

Uma ferramenta de simulação em tempo real que replica — em JavaScript puro — todo o pipeline de cálculos que um software como Aspen Plus ou HYSYS faria para uma planta de biometano.

⚡

Tempo Real

Cada slider atualiza instantaneamente todos os cálculos — balanços, dimensionamentos, custos. Sem esperar simulação.

✅

Validado

Calibrado contra o pipeline Python do grupo (ecogas_v4) com 0% de erro nos resultados-chave. 7 sanity checks automáticos.

📚

Didático

Cada número mostra a fórmula, a referência bibliográfica e a faixa de validade. Feito para ensinar e defender na banca.

2. Pipeline Cascata (Modelo Computacional)

O cálculo segue a sequência exata do processo industrial — cada bloco alimenta o próximo:

MX-01→ BIO-01→ HE-101→ K-101→ HE-102→ C-101 HPWS→ S-101→ CHP-01
MX-01: Mistura de 3 substratos (lodo 80% + vinhaça 10% + resíduo 10%). Balanço de massa e composição elementar.
BIO-01: 4 biorreatores CSTR. Cinética de Haldane dual-pathway (acetoclástica + hidrogenotrófica). TRH = 28.8 d.
HE-101: Trocador casco-tubo. Aquece substrato de 25→35°C com água quente do CHP. LMTD, Uo, Re, ΔP.
K-101: Compressor 2 estágios politrópico (n=1.3). P1=1atm → P2=10bar abs. Inter-resfriamento em HE-102.
C-101 HPWS: Coluna de absorção. Onda 1968 (kL/kG), NTU/HTU, GPDC flooding, Henry-Sander. Q_água = 81.63 m³/h.
S-101: Dessecador PSA (peneira molecular). Ponto de orvalho -40°C. ANP 685/2017 compliance.
CHP-01: Cogeração (η_el=40%, η_th=45%). Fecha o loop térmico do digestor. Produz 354 kW elétrico.
Hidráulica: 13 trechos com Colebrook-White. 8 bombas (P-101 a P-108). 18 válvulas especificadas.

3. Abas do Dashboard — O Que Cada Uma Mostra

14 abas cobrindo desde o PFD interativo até análise econômica com VPL/TIR:

🏭

PFD Interativo

Fluxograma de processo vetorial (SVG). Clique em qualquer equipamento para ver dimensionamento, material, norma e fórmulas. Linhas animadas mostram o fluxo.

⚖

Balanço de Massa

Conservação de massa com erro < 0.01%. Tabela de alimentação, biogás, digestato, scrubber. Diagrama Sankey de massa. Tudo reativo aos sliders.

🔥

Balanço de Energia

1ª Lei da Termodinâmica. PCI do CH₄, potência mecânica (bombas + compressor), demanda térmica, eficiência energética global. Sankey energético.

💧

Hidráulica

13 linhas de processo com Swamee-Jain/Colebrook (±1%). Velocidade, Reynolds, fator de atrito, perda de carga distribuída e localizada.

⚓

Bombas (P-101 a P-108)

Dimensionamento completo: Hm, potência instalada, Reynolds, regime (7/8 turbulento), NPSHd, tipo/material/norma. P-106 domina com 48 kW (95%).

🛑

Válvulas (18 unidades)

Especificação por ISA S75.01, API 6D, API 520. Gaveta, esfera, globo, borboleta, retenção, segurança. Cv, classe de pressão, material.

🧪

HPWS Rigorosa

Coluna de absorção com correlação de Onda 1968 (kL/kG), integração numérica NOG (2000 pts), GPDC, slip CH₄, flash recovery, verificação isotérmica.

🔥

Trocadores de Calor

HE-101 (pré-aquecimento) e HE-102 (intercooler). Método de Kern: LMTD, Uo, número de tubos, Re, ΔP, gargalo térmico identificado.

📈

Gráficos + Otimização + Econômico

Cenários A-E, sensibilidade 1D, superfície 3D, VPL/TIR/Payback com taxa ajustável. Distribuição CAPEX/OPEX em gráficos interativos (Plotly).

5. Diferencial Competitivo

O que separa este dashboard de uma planilha Excel ou de um relatório estático:

Método Tradicional

  • ❌ Planilhas isoladas, sem integração
  • ❌ Mudar um parâmetro exige recalcular tudo manualmente
  • ❌ Sem validação automática de coerência física
  • ❌ Resultados estáticos (PDF/impresso)
  • ❌ Difícil de auditar — "de onde veio esse número?"

Dashboard ECOGAS

  • ✅ Pipeline cascata integrado (cada bloco alimenta o próximo)
  • ✅ Sliders recalculam tudo em <50 ms
  • ✅ 7 sanity checks automáticos + verificação de Reynolds
  • ✅ Interativo — explore cenários em tempo real
  • ✅ Cada número mostra fórmula + referência no tooltip

6. Como Utilizar

Passo a passo para explorar o dashboard:

01

Abra o Dashboard

Acesse via GitHub Pages ou abra index.html localmente. Funciona 100% offline (sem servidor).

02

Explore as Abas

Comece pelo PFD para visão geral. Clique nos equipamentos para ver dimensionamento. Cada aba é independente.

03

Ajuste os Sliders

Clique em "⚙ Premissas" no canto superior direito. Mova os sliders de TRH, vazão, frações — veja o impacto instantâneo.

04

Verifique Coerência

Os badges no header mostram os sanity checks. Verde = coerente. Vermelho = inconsistência (ex: violação de conservação de massa).

05

Compare Cenários

Na aba Otimização, compare cenários A-E ou salve seus próprios. Na aba Econômico, veja VPL/TIR/Payback com taxa ajustável.
